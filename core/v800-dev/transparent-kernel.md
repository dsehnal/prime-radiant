## Recurrence forms

Expanded:

`Perspective(n) = Interpret(Context(n-1))`
`Context(n) = Integrate(Context(n-1), Perspective(n))`

Collapsed:

`Context(n) = Integrate(Context(n-1), Interpret(Context(n-1)))`

Compact symbolic shadow:

`C_n = ∫ C_(n-1) ∇C_(n-1)`

with the understanding that this is evocative / compressed notation, not literal standard calculus.

## Remainder / Shield family

Current best distinction:

* **Remainder** = unresolved structure in general
* **Residue** = a named remainder preserved for carry-forward

So the transparent theory can say something like:

> A residue is a named remainder carried into the next cycle.

This preserves the idea that what used to be called a Shield is a stabilized subset/form of remainder rather than a separate primitive.
