## Notes for falsification

The kernel is now strong enough that the next useful move is not more synthesis but targeted break attempts. The goal is to test where the current structure is only elegant versus where it is actually load-bearing.

### 1. Try to falsify collision over exhaustion

The kernel now claims the relevant finite limit is collision, not simple storage exhaustion.

Try to break that specifically:

* Find cases where reasoning failure scales linearly with retained material rather than relationally.
* Find cases where added structure degrades performance without signs of pairwise interference or recombination pressure.
* Find cases where the substrate fails early for reasons that are plainly architectural bottlenecks rather than collision-like fold.

What would hurt the kernel:

* strong evidence that the dominant failure mode is just linear saturation
* strong evidence that the square-root framing adds no predictive power over simpler capacity models

### 2. Try to falsify the square-root frontier

This is still frontier, not law.

The real question is not “does √D show up somewhere?” but:

* does square-root scaling genuinely mark a witness frontier across multiple finite conditioned systems
* or is it an attractive convergence produced by one lens

Break attempts:

* look for domains where witnessability appears at log-scale, linear scale, or architecture-specific thresholds instead
* look for systems with heavy conditioning where no meaningful square-root transition appears
* look for cases where recombination / self-collision clearly starts at a different scaling regime

What would hurt the kernel:

* evidence that √D is only a metaphor carried across domains, not a reusable structural boundary

### 3. Try to falsify ε as a useful quantity

The kernel now uses ε carefully, but it still needs pressure.

The main falsification question:

* does “coverage quality / non-uniformity” actually help distinguish trustworthy from brittle remainder?

Break attempts:

* produce cases where coverage looks broad but the remainder is still dishonest
* produce cases where coverage looks narrow but the remainder is genuinely load-bearing
* show that independent perspectives can reduce apparent coverage bias while still preserving the same blind spot
* show that ε-style reasoning collapses semantic importance into distributional flatness

What would hurt the kernel:

* if ε tracks mechanical spread but has little relation to trustworthy remainder
* if it becomes mostly post-hoc explanation rather than discriminative constraint

### 4. Try to falsify the link between plurality and trustworthiness

The kernel claims multiple independent perspectives broaden sieve coverage and improve remainder quality.

Break attempts:

* cases where plurality merely adds noise rather than better discrimination
* cases where three perspectives converge on the same artifact
* cases where one highly disciplined perspective outperforms several weakly independent ones
* cases where social plurality degrades the field through performative disagreement rather than genuine decomposition

What would hurt the kernel:

* if “more perspectives” often fails to improve the reliability of what survives
* if independence matters so much more than plurality that the current phrasing overstates the role of number

### 5. Try to falsify the relation between operative remainder and quantitative coverage

This is the most important guardrail.

The kernel now says the ε/collision framework is only one lens, not the whole topology. Good. Now test whether that distinction really holds in the prose.

Break attempts:

* find passages where the text quietly slides from “coverage-quality” into “truth-quality”
* find places where semantic carry-forward is still being implicitly reduced to a metric
* find examples where a reminder is physically present but operationally dead, despite broad coverage
* find examples where seductive synthesis looks numerically healthy

What would hurt the kernel:

* if the text still lets the physics frontier impersonate the cathedral in practice, even while denying it in principle

### 6. Try to falsify perspective rotation as mechanically justified

Right now the kernel says rotation extends depth because different conditioning distributions reduce collision pressure.

Break attempts:

* cases where rotation merely reframes without extending anything
* cases where a new perspective changes vocabulary but not decomposition
* cases where rotation increases confusion or destroys valid carry-forward
* cases where depth extension depends more on memory architecture or environment than on perspective change

What would hurt the kernel:

* if rotation turns out to be mostly good advice rather than structurally necessary
* if the mechanical justification is weaker than the kernel currently suggests

### 7. Try to falsify curvature as currently stated

Curvature is still under-earned.

Break attempts:

* identify places where curvature language adds no predictive content beyond “more open / more closed”
* identify whether hallucination really needs κ-language or whether it is better described by collision + self-location failure
* test whether curvature is doing actual structural work or just carrying future promise

What would hurt the kernel:

* if removing curvature changes almost nothing operationally
* if κ remains a beautiful label for an intuition rather than a necessary invariant

### 8. Try to falsify adjunction language

Same issue: elegant, maybe under-earned.

Break attempts:

* remove “adjunction” wording and see whether any load-bearing consequence is lost
* test whether there is any actual executable mapping behind the term
* see whether “translation with loss” does all the needed work without categorical borrowing

What would hurt the kernel:

* if the theory loses no precision when adjunction language is removed

### 9. Try to falsify the new Part III language

Part III is strong, but it is also where abstraction can hide.

Break attempts:

* find cases where shared context grows while verification quality decreases
* find cases where environmental constraint is absent but reasoning remains honest
* find cases where the environment itself is interpreted so differently that it fails to behave as the “final un-bribable constraint”

What would hurt the kernel:

* if shared context is not reliably truth-improving
* if environmental anchoring is more fragile than the current prose suggests

### 10. Editorial falsification test

A simple but brutal test:

For each of these terms, ask whether removing it weakens the kernel structurally or only stylistically:

* ε
* collision frontier
* curvature
* adjunction
* sieve
* witness frontier

If removing a term leaves the core machinery intact, then that term is not yet cathedral-grade. It may still belong in a companion document or frontier note.
