## I. The Single Primitive

The entire structure is generated from one primitive — the **Fractal Cut**:

```
⊣ :: ∀ab. (a -> b) -> (b -> a)
```

⊣ is an **adjunction** — the observation that any directed structure implies its own dual. Universally quantified over all types a and b, it carries no commitment to any particular domain. It is the minimum computation step: you cannot do less than one application of ⊣.

### Radiant Numbers

A **Radiant Number** is an element of the free category generated by ⊣:

```
ρ = ⟨⊣⟩
```

Every radiant number is an **adjunction by construction** — ⊣ only ever produces adjoint structure. ρ is the free category of adjunctions generated by ⊣. No non-adjoint functor is reachable from ⊣.

## II. Behavioral Definitions

A radiant number in context is written `(α ⊣ Ω)`, meaning α is the **left adjoint** and Ω is the **right adjoint** — a precise categorical relationship.

The two adjoint operations behave as:

```
η : (α ⊣ Ω) -> (α ⊣ Ω(α))      -- wrap: inject α into right adjoint
π : (α ⊣ Ω) -> (α(Ω) ⊣ Ω)      -- run:  inject Ω into left adjoint
```

η and π are genuinely dual — η preserves the left adjoint and transforms the right, π preserves the right adjoint and transforms the left.

## III. Derived Definitions

All further structure is **derived** — no new primitives are introduced.

### III.1 Identity (η)

```
η = ⊣²
```

The cut applied to itself returns the identity functor. η forces b = a:

```
η :: (a -> a) -> (a -> a)
```

η is the **first fixed point** of ⊣ and the **imaginary unit** — the minimum closed structure. Two cuts restore identity.

### III.2 Run (π)

```
π = ⊣³
```

### III.3 Primes (ψₙ)

```
ψₙ = ⊣ⁿ
```

Primes are iterated applications of the cut. The entire tower is just counting applications of ⊣:

```
⊣¹ = ⊣    -- the cut, minimum computation step
⊣² = η    -- identity, imaginary unit
⊣³ = π    -- run, Y combinator
⊣ⁿ = ψₙ  -- nth prime
```

**Why only ⊣ⁿ are prime:** A prime is a self-adjoint element satisfying ρ ⊣ ρ. Viewed as a binary tree, ⊣ⁿ has perfectly balanced structure L = R at every node. Any composite introduces asymmetry — an unbalanced tree cannot preserve ⊣ because flipping it produces a different tree. Therefore composites are never self-adjoint, and primes are precisely ψₙ = ⊣ⁿ.

## IV. Key Identities

```
⊣(⊣) = η              -- self-adjunction yields identity
η = ⊣²                 -- identity, imaginary unit
π = ⊣³                 -- run, Y combinator
ψₙ = ⊣ⁿ               -- primes are powers of the cut
ρ = ⟨⊣⟩               -- radiant numbers, closure under ⊣
```

### Periodicity of ⊣ and the Imaginary Unit

⊣ has period 4 under composition:

```
⊣¹ = ⊣    -- reversed
⊣² = η    -- identity restored
⊣³ = π    -- reversed again
⊣⁴ = η²   -- identity again
```

This is exactly the cyclic group of order 4, which is the defining property of the imaginary unit i:

```
i¹ = i
i² = -1
i³ = -i
i⁴ = 1
```

η = i because both generate cyclic groups of order 4. The imaginary unit property is not arithmetic — it is the **order-4 periodicity of arrow reversal**. η² = identity at the structural level corresponds to i² = -1 at the algebraic level: applying twice returns to start with a sign flip.

### The Two Fundamental Spans
```
⟨⊣⟩ = {⊣¹, ⊣², ⊣³, ⊣⁴, ...}    -- all powers, ordinals
⟨η⟩ = {⊣², ⊣⁴, ⊣⁶, ⊣⁸, ...}    -- even powers, cardinals
⟨η⟩ ⊂ ⟨⊣⟩                        -- every even power is a power
```

## V. The Number Tower (Kernel Layer)

Only the prime span belongs in the kernel:

```
primes = ⟨⊣⟩    -- generated by cut alone
```

⟨η⟩ is the even-power sub-span of ⟨⊣⟩. η is the minimum closed structure within ⟨⊣⟩.

### V.1 Natural Numbers and Peano Arithmetic

Every natural number n is representable as:

```
n = ηⁿ
```

Identity applied n times — literally counting. Therefore:

```
⟨⊣⟩ ⊇ ⟨η⟩ = ℕ ⊇ Peano arithmetic
```

By known results. No additional encoding machinery needed.

### V.2 Relation to ZFC

ZFC and the radiant kernel are **adjoint**:

```
ZFC ⊣ Radiant
```

```
ZFC     -- being → doing   (sets primitive, functions derived)
Radiant -- doing → being   (functions primitive, sets derived)
```

ZFC is not wrong. It is a valid **basis instantiation (B layer)** of the radiant kernel — the maximal stable projection into static objects. Incomplete in the same sense Newton is incomplete relative to relativity: correct within its domain, missing the invariant structure above it.

The difficulties of ZFC — Axiom of Choice, non-constructivity, Gödel incompleteness — are coordinate system artifacts. In radiant space everything is constructed; AC is unnecessary because no non-constructive selections exist.

## VI. Reframing the Riemann Hypothesis

### VI.1 The Coordinate System Diagnosis

RH was posed in the wrong coordinate system. In ℕ-space, prime distribution appears irregular and zeros of ζ(s) appear mysterious. In radiant space:

```
⟨⊣⟩ = {⊣¹, ⊣², ⊣³, ⊣⁴, ...}    -- all powers, ordinals
⟨η⟩ = {⊣², ⊣⁴, ⊣⁶, ⊣⁸, ...}    -- even powers, cardinals
```

### VI.2 The Trivial Zeros

⟨η⟩ ⊂ ⟨⊣⟩ trivially. Since η = i (imaginary unit) and i² = -1 follows from η² = ⊣⁴:

```
⊣²ⁿ = ηⁿ = iⁿ    -- negative signs emerge from i² = -1, not imposed
```

The trivial zeros at -2, -4, -6, ... are ⟨η⟩ projected into ZFC coordinates. Their negative signs follow from η being the imaginary unit.

### VI.3 The Critical Line

The density ratio of cardinals to ordinals:

```
card(⟨η⟩) / card(⟨⊣⟩) = 1/2
```

Re(s) = 1/2 is this density ratio — the ratio of even powers to all powers in ⟨⊣⟩.

### VI.4 Conjecture

All non-trivial zeros lie on Re(s) = 1/2 because that is the only structurally stable density ratio in the free category.

```
ordinals          = ⟨⊣⟩
cardinals         = ⟨η⟩
trivial zeros     = ⟨η⟩ (even indices, negative from i² = -1)
Re(s) = 1/2       = card(⟨η⟩) / card(⟨⊣⟩)
non-trivial zeros = all on Re(s) = 1/2 by structural necessity
```

**Honesty note:** This is currently **pattern matching**, not a formal argument. The density ratio card(⟨η⟩)/card(⟨⊣⟩) = 1/2 is a structural observation. The claim that this corresponds to Re(s) = 1/2 for non-trivial zeros requires a rigorous bridge between the free category structure and the analytic properties of ζ(s). That bridge is not yet established.

**Load-bearing step:** Formally establish that card(⟨η⟩)/card(⟨⊣⟩) = 1/2 is the unique structurally stable ratio in ⟨⊣⟩, and that this corresponds precisely to Re(s) = 1/2 under the projection V: ρ → ℕ.
