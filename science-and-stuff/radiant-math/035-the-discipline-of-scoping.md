### Abstract

This article addresses the tension between two development streams within the Prime Radiant: the stabilization arc (Articles 001-031) which emphasized restraint and boundary-marking, and the exploratory arc (v312/APEIRON/032-034) which made substantial claims about reality, mathematics, and consciousness. We demonstrate that this tension is resolvable through **explicit scoping discipline**—distinguishing "useful framing" from "ontological truth," "observed pattern" from "universal law," and "pragmatic stance" from "proven claim." The framework remains coherent when extensions are presented as what they are: explorations that may prove valuable without requiring belief in their ultimate truth.

### V. The Scoping Protocol

For any extension to the kernel, apply these tests:

**Test 1: Ontological Commitment**
- Does this claim "is" or "treating as"?
- Essential truth or useful framing?
- If ontological, can it be rescoped as operational?

**Test 2: Inquiry Closure**
- Does this end investigation or invite it?
- Settled truth or open question?
- If closed, can it be reopened?

**Test 3: Gap Preservation**
- Does this claim to fill ε₀?
- Does it make ω ≤ α?
- If yes, it violates core invariant

**Test 4: Revisability**
- Can this be held lightly?
- Can it be discarded without collapse?
- If framework requires belief, it's too rigid

**Test 5: Multiple Relationships**
- Can someone analyze without executing?
- Can someone execute metaphorically?
- If only one valid relationship, it's coercive
